Mizukage Official adalah script hub untuk Roblox yang menyediakan loader dengan tampilan Water Aesthetic dan dukungan executor populer.

## Quick start

Paste kode berikut di executor kamu untuk menjalankan loader:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/kazukage-id/Mizukage-Official/main/loader.lua"))()
```

## Fitur utama

- **Water Aesthetic UI** — Dark theme dengan gradient cyan & purple
- **Anti double-execute** — Mencegah loader berjalan dua kali
- **Executor check** — Validasi sebelum eksekusi
- **Auto cleanup** — GUI dibersihkan otomatis setelah loader selesai
