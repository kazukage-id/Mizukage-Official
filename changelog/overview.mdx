## Rekap mingguan — Script Poop Game

Script per-game baru **Poop Game** resmi tersedia di katalog Mizukage Official. Berjalan otomatis lewat Cloud Launcher v16.0 atau loader v2.0 di executor besar (Synapse Z, Fluxus, Delta, Hydrogen, Arceus X).

### Fitur baru

- **Auto Farm Poop** — Rutin balancing otomatis menargetkan zona aktif sehingga sesi farming berjalan tanpa input manual.
- **Auto Sell Inventory** — Inventori dijual berkala dengan interval yang dapat diatur (1–180 detik) untuk menjaga slot tetap kosong.
- **Instant Poop** — Mempercepat siklus poop dengan interval konfigurabel (mulai 0.30 detik).
- **Remove Poops** — Membersihkan objek poop di workspace untuk mengurangi lag, tanpa mengganggu NPC penjual.
- **Player tweaks** — Toggle **Noclip** dan **WalkSpeed Changer** (28–100) untuk mobilitas penuh selama farming.
- **Misc tools** — **Anti AFK**, **Anti Fling**, dan **Touch Fling** tersedia langsung dari panel kontrol.
- **Panel WindUI** — GUI dengan tab **Main**, **Player**, **Exploit**, dan **Misc** mengikuti tema Water-Aesthetic milik loader v2.0.

### Update

- **Konfigurasi global lewat `MizuConfig`** — Semua toggle Poop Game dapat diatur ulang dari sesi loader yang sama, konsisten dengan script Mizukage lainnya.
- **Guard anti double-load** — Pemanggilan ulang script saat sesi aktif diabaikan agar konfigurasi yang sedang berjalan tidak tertimpa.
- **Kompatibilitas executor** — Mengikuti dukungan loader v2.0; daftar lengkap tersedia di [halaman utama](/).

</Update>

## Rekap mingguan — Script Lemon Sells v2

Script per-game baru resmi tersedia di katalog Mizukage Official. Cocok dijalankan lewat Cloud Launcher v16.0 atau loader v2.0 di semua executor besar.

### Fitur baru

- **Lemon Sells v2 (Base Universal)** — Modul otomatisasi tycoon Lemon dengan tiga rutin utama: **Auto Buy** (membeli upgrade/tombol yang tersedia), **Auto Upgrade**, dan **Auto Fruit**. Sesi tetap berjalan di latar belakang tanpa GUI tambahan.
- **Guard anti double-load** — Saat script sudah aktif, pemanggilan berikutnya menampilkan notifikasi *“Sistem Lemon sudah berjalan!”* sehingga sesi yang berjalan tidak tertimpa.
- **Notifikasi tycoon tidak ditemukan** — Bila tycoon milik pemain belum tersedia di server, script memberi notifikasi jelas alih-alih gagal diam-diam.

### Update

- **Konfigurasi global lewat `MizuConfig`** — Status `Buy`, `Upgrade`, `Fruit`, dan `Delay` dapat diatur ulang dari sesi loader yang sama, memudahkan pengguna mengaktifkan/mematikan rutin tanpa menjalankan ulang snippet.
- **Kompatibilitas executor** — Mengikuti dukungan loader v2.0; berjalan di Synapse Z, Fluxus, Delta, Hydrogen, dan Arceus X. Lihat daftar lengkap di [halaman utama](/).

</Update>

## Rekap mingguan — Loader produksi resmi tayang

Build produksi `loader.lua` kini diterbitkan ke kanal distribusi utama. Snippet `loadstring` di [halaman utama](/#eksekusi-utama) sudah mengarah ke build final yang telah diobfuscate, sehingga sesi loader siap dipakai langsung dari executor tanpa langkah tambahan.

### Update

- **Loader produksi diobfuscate** — Build resmi diterbitkan dengan proteksi obfuscation untuk mengurangi risiko *reverse engineering* dan duplikasi script tidak resmi.
- **Kanal distribusi utama aktif** — `loader.lua` dapat diakses langsung melalui snippet `loadstring` di [Eksekusi utama](/#eksekusi-utama); kanal cadangan `scripts/main.lua` tetap tersedia bila kanal utama bermasalah.

### Perbaikan bug

- **Konsistensi versi loader** — Snippet di dokumentasi dan build di kanal distribusi kini berasal dari sumber yang sama, sehingga tidak ada lagi selisih versi antara contoh di dokumentasi dan loader yang dieksekusi.

</Update>

## Cloud Launcher v16.0 — Auto-Execute & Webhook Log

Mizukage Official Cloud Launcher v16.0 (Minimal Edition) resmi tersedia. Versi ini fokus pada eksekusi script per-game secara otomatis tanpa GUI, cocok dipasangkan dengan loader v2.0 untuk pengguna PC maupun mobile.

### Fitur baru

- **Cloud Launcher per-game** — Launcher kini menarik database game dari kanal cloud resmi dan otomatis memuat script yang sesuai dengan `PlaceId` aktif, jadi Anda tidak perlu mengganti snippet `loadstring` setiap pindah game.
- **Auto-Execute setelah teleport** — Sesi launcher bertahan saat berpindah server atau game, sehingga script langsung berjalan kembali tanpa eksekusi ulang manual.
- **Webhook Log eksekusi** — Setiap sesi launcher melaporkan status eksekusi ke Discord melalui webhook resmi, memudahkan pemantauan sesi aktif. Lihat snippet [Alternative URL](/#eksekusi-utama) untuk memakai endpoint cadangan saat kanal utama bermasalah.

### Update

- **Dukungan multi-executor diperluas** — Launcher otomatis mendeteksi fungsi `request` dari Synapse, Fluxus, Hydrogen, dan executor besar lain, sehingga snippet yang sama bekerja di semua executor yang didukung. Daftar lengkap di [halaman utama](/).
- **Guard anti double-launch** — Sesi launcher aktif dilindungi dari pemanggilan ganda, mengurangi risiko crash saat snippet dipanggil dua kali berturut-turut.

### Perbaikan bug

- **Notifikasi executor tidak kompatibel** — Pengguna executor tanpa `HttpGet`/`loadstring` kini menerima pesan jelas dari launcher, bukan gagal diam-diam.

</Update>

## Peluncuran Mizukage Official Loader v2.0

Loader resmi Mizukage Official kini tersedia untuk publik. Rilis perdana ini membawa pengalaman eksekusi script yang lebih ringan, aman, dan stabil di seluruh executor besar.

### Fitur baru

- **Aesthetic UI Loader v2.0** — Antarmuka berbasis WindUI dengan tema Water-Aesthetic (gradient Cyan/Purple) dan animasi transisi *Easing Styles*. Cek pratinjau di [halaman utama](/).
- **Alternative loader URL** — Snippet `loadstring` cadangan dari channel `scripts/main.lua` untuk dipakai saat `raw.githubusercontent` sedang down. Lihat tab *Alternative URL* di [Eksekusi utama](/#eksekusi-utama).
- **Webhook Log gratis** — Log eksekusi via webhook yang biasanya berbayar kini disediakan tanpa biaya tambahan.
- **Auto Reconnect & Anti-AFK lanjutan** — Sesi tetap aktif otomatis tanpa konfigurasi manual.

### Update

- **Kompatibilitas executor diperluas** — Dukungan resmi untuk Synapse Z, Fluxus, Delta, Hydrogen, dan Arceus X. Daftar lengkap di [halaman utama](/).
- **Proteksi keamanan ditingkatkan** — Enkripsi dinamis, validasi HWID, dan proteksi Anti-Dump aktif secara default di setiap sesi loader.
- **Guard anti double-execute** — Loader otomatis mendeteksi sesi aktif dan mencegah eksekusi ganda yang sering menyebabkan crash.

### Perbaikan bug

- **Notifikasi executor tidak kompatibel** — Pengguna executor tanpa dukungan `HttpGet`/`loadstring` kini menerima notifikasi yang jelas, bukan gagal diam-diam.

</Update>
