Loader Script — Cara Pakai

Copy satu baris ini ke executor kamu. Loader akan tampil otomatis dengan animasi water-aesthetic, lalu mengeksekusi script utama kamu.

ℹ️
Pastikan executor mendukung HttpGet Loader menggunakan game:HttpGet() untuk mengambil script dari GitHub. Cek bagian Executor untuk daftar yang didukung.
Lua — Execute di Executor
loadstring(game:HttpGet("https://raw.githubusercontent.com/NAMA_KAMU/mizukage/main/loader.lua"))()
✅
Berhasil! Animasi loader Mizukage akan muncul selama ~4 detik, lalu script utama berjalan otomatis.
✨
Urutan Animasi Loader

Loader melewati 8 fase animasi dari muncul hingga menghilang sebelum script dieksekusi.

1
Overlay Fade-In
Layar ditutupi overlay semi-transparan gelap. Durasi 0.5 detik.
2
Card Pop-In
Card loader muncul dengan animasi Back.Out dari titik tengah layar. Durasi 0.55 detik.
3
Reveal Text & Bar
Logo kanji 水, judul, subtitle, dan progress bar fade-in berurutan.
4
Pulse Loop
Glow di belakang logo berdenyut terus selama proses loading berjalan.
5
Loading Steps (×4)
Status text berganti dengan fade, progress bar meningkat: 20% → 40% → 60% → 85% → 100%.
6
Flash Accent
Card berkilat cyan sesaat sebagai tanda loading selesai.
7
Outro — Semua Fade-Out
Semua elemen menghilang, card mengecil kembali ke titik 0 dengan animasi Back.In.
8
Eksekusi Script Utama
ScreenGui di-destroy lalu loadstring(game:HttpGet(URL))() dijalankan dalam pcall.
🔒
Sistem Keamanan Bawaan

Loader sudah dilengkapi beberapa lapisan proteksi out-of-the-box.

🚫
Anti Double-Execute
Flag getgenv().MizukageLoaderActive mencegah loader berjalan dua kali dalam satu sesi.
🔍
Executor Check
Verifikasi game.HttpGet dan loadstring tersedia sebelum loader berjalan.
🛡️
pcall Protection
Eksekusi script utama dibungkus pcall — error ditangkap dan ditampilkan sebagai notifikasi.
🧹
Auto Cleanup
ScreenGui dan flag global dihapus otomatis setelah loader selesai agar tidak ada memory leak.

Konfigurasi loader.lua

Edit bagian ini di loader.lua untuk menyesuaikan dengan project kamu.

Lua — Variabel yang Perlu Diedit
-- ⬇️ EDIT BAGIAN INI SESUAI PROJECT KAMU -- Warna accent utama (default: Cyan) local ACCENT = Color3.fromRGB(0, 255, 255) -- Warna accent kedua (default: Purple) local ACCENT2 = Color3.fromRGB(100, 80, 255) -- ⬇️ PALING PENTING: URL script utama kamu local MAIN_SCRIPT_URL = "https://raw.githubusercontent.com/NAMA_KAMU/mizukage/main/scripts/main.lua"
VariabelDefaultKeterangan
ACCENTCyanWarna utama UI (progress bar, teks judul)
ACCENT2PurpleWarna gradient kedua
MAIN_SCRIPT_URL—Wajib diisi — link raw GitHub script utama
Teks Logo水Ganti di bagian Text = "水" pada Logo label
Teks JudulMIZUKAGE OFFICIALGanti di bagian Text = "MIZUKAGE OFFICIAL"

Deploy ke GitHub Pages

Dokumentasi ini bisa langsung dihosting gratis di GitHub Pages dengan beberapa langkah.

1
Buat Repository Baru
Pergi ke github.com → New Repository → Nama: mizukage → Public → Create.
2
Upload Semua File
Upload loader.lua, folder scripts/, dan docs/index.html ke repo tersebut.
3
Aktifkan GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Folder: /docs → Save.
4
Ganti NAMA_KAMU
Edit loader.lua — ganti semua NAMA_KAMU dengan username GitHub kamu, lalu commit.

FAQ
Q
Loader tidak muncul sama sekali?
Pastikan executor kamu mendukung game:HttpGet(). Cek error di console executor dengan melihat output warn().
Q
Notifikasi "Gagal memuat script utama" muncul?
Berarti MAIN_SCRIPT_URL di loader.lua belum diisi atau URL salah. Pastikan link raw GitHub benar dan repo public.
Q
Cara ganti warna loader?
Edit variabel ACCENT dan ACCENT2 di bagian [2] file loader.lua menggunakan Color3.fromRGB(R, G, B).
Q
Loader muncul dua kali setelah rejoin?
Normal — flag getgenv() direset saat rejoin. Loader hanya dicegah dobel dalam satu sesi tanpa rejoin.
